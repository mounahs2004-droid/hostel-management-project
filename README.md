# Hostel-management-project
Hostel management system developed using python and database

Project Description

The Hostel Management System is designed to manage hostel activities such as student registration, room allocation, fee management, and record maintenance.
This system reduces manual work and helps administrators manage hostel data efficiently.

Objectives

To maintain student hostel records digitally
To manage room allocation and availability
To track hostel fees and payments
To reduce paperwork and improve accuracy

Technologies Used
Programming Language: Python
Database: MySQL / SQLite
Tools: VS Code
Version Control: Git & GitHub

2.Navigate to the project folder
cd hostel-management-system
3.Install required packages
pip install -r requirements.txt
4.Run the application
python main.py
